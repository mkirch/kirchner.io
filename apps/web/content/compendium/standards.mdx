---
title: Data Formats, Specifications, and Standards
description: Data formats, specifications, and standards exist. This page lists some of them and their obscene number of acronyms.
icon: 'file-shield'
---

- [RFC 8259](https://datatracker.ietf.org/doc/rfc8259/)
- [IETF Data Tracker](https://datatracker.ietf.org/)
- [IANA Media Types](https://www.iana.org/assignments/media-types/media-types.xhtml)
- [IETF](https://www.ietf.org/): The home of the Internet Engineering Task Force (IETF), the body that develops and promotes Internet standards.
- [RFC 1](https://www.rfc-editor.org/rfc/rfc1): The first RFC, published in 1969. It is a request for comments on the topic of Host Software. It is a short document, and it is interesting to read it and see how the Internet has evolved since then.
- [RFC History](https://www.rfc-editor.org/rfc/rfc8700.html#name-key-moments-in-rfc-history): Key moments in RFC history.
- [JSON-LD Specification](https://www.w3.org/TR/2020/REC-json-ld11-20200716/#bib-rfc8259): The JSON-LD spec, which references RFC 8259, the JSON spec. JSON-LD is a JSON format for Linked Data, which is a way of representing data that is similar to the way the Web is structured.
- [Unicode](https://home.unicode.org/)
- [INCITS (InterNational Committee for Information Technology Standards)](https://www.incits.org/)
- [ISO.org](https://www.iso.org/home.html)
- [Compart](https://www.compart.com/)
